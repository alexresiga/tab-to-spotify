# tanananify
Google Chrome extension to get [TANÄ‚NANA](http://tananana.ro/) radio playing queue and add tracks to your Spotify library

## how to use

`git clone https://github.com/alexresiga/tab-to-spotify.git`

open `Google Chrome` and enter `chrome://extensions`

enable `Developer mode`

click `Load unpacked`

browse to clone location and select folder

enjoy
